Mid sized react based project aimed at showcasing better knowledge of statemanagement, effects, hooks, refs and other custom features. The project is
in large part a movie DB app that allows the user to search for their favourite movies, find said movie rate it and add it to a watched list

## APP Features

This app utilizes several features that would typically be found in a movie db app

- Two main box elements to showcase searched movies and movie details / a watched movies panel
- The ability to search for their favourite movies and add them to a watched list
- A rating system to allow the user to keep track of their rated movies
- Ability to delete movies from the list
- Saving the list to local storage so that the user can view their list on fresh launch
- back end tracking of how many times they rated a specific movie

## Installation Guide

First clone this repo

```sh
git clone https://github.com/ShaAnder/react-popcorn.git
```

NPM install to get the dependancies

```sh
npm install
```

Use npm start to start the local server / test the app

```sh
npm start
```
